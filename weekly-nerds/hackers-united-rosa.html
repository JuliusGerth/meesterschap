Weekly Nerd #5

Hackers United Rosa

Digitale Onafhankelijkheid & The Travelling Server

Wat is een hacker?

Een hacker is iemand die systemen begrijpt en creatief gebruikt — niet per se voor kwaad, maar om dingen te repareren, hergebruiken of verbeteren op manieren die de maker niet bedoeld heeft.

Steeds moeilijker

Planned obsolescence werkt reparatie en hacken actief tegen. Producten worden zo gemaakt dat ze na een bepaalde tijd niet meer goed functioneren, met als enige oplossing: nieuw kopen. Dit vergroot de afhankelijkheid van producenten.

Grotere afhankelijkheid van Big Tech

We zijn steeds afhankelijker van grote techbedrijven — niet alleen als consumenten, maar ook als producenten. Zelfs overheden vertrouwen sterk op de infrastructuur van een handvol bedrijven. Dit creëert risico's op het gebied van privacy, controle en continuïteit.

Waar leeft je website?

In "de cloud" — maar dat zijn gewoon datacenters van andere bedrijven. De vraag is: wie controleert die datacenters, en wat gebeurt er als ze verdwijnen?

Rosa the Travelling Server

Rosa laat zien dat het anders kan. Door te werken met oude hardware en open-source software kun je zelf de controle houden over je infrastructuur.

Oude hardware heeft invloed op de keuzes die je maakt voor je frameworks en je HTML, CSS en JavaScript. Dit dwingt je tot het maken van lichtere, duurzamere keuzes.

Static Site Generators

Voor een server met beperkte capaciteit zijn static site generators een goede keuze. Ze genereren HTML-bestanden die direct kunnen worden geserveerd zonder server-side verwerking:

  • Minder impact op de server
  • Als alles faalt, heb je nog altijd de Markdown-bestanden — leesbare code
  • 11ty is een goede keuze: grote community en veel documentatie

Zelf hosten met open source

Met tools zoals YunoHost of Linux kun je zelf een server beheren en allerlei diensten hosten:

  • Git (Gitea / Forgejo)
  • Messaging platform (Zulip, Libera Chat)
  • Je eigen website
  • Een radioshow

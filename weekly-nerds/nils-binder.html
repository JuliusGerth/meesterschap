Van Photoshop naar Figma naar AI

Vroeger gebruikte iedereen Photoshop om een website te ontwerpen, nu is er een overstap gemaakt naar Figma. Figma lijkt meer op hoe je met CSS werkt waardoor designers minder onmogelijke dingen ontwerpen. De nieuwe "holy triangle van saai" is: Figma → Tailwind → AI.

Tip: kijk niet naar pixels maar naar de verhouding tussen elementen. Gebruik hiervoor een grid.

CSS technieken

CSS Grid met fr units

.class {
    display: grid;
    grid-template-columns: 2fr 3fr auto 1fr;
}

h1 {
    grid-column: 2 / 4;
}

p {
    grid-column: 3 / 4;
}

Het grid past zich aan aan de grootte van het scherm.

Stop-motion animatie

animation: stop-motion 3s steps(31, end) infinite;

De animatie is opgebouwd uit meerdere afbeeldingen in een lange strip met 31 fases. Door steps() te gebruiken in plaats van linear krijg je een stop-motion effect.

Mix-blend-mode

mix-blend-mode: difference;

Inverteert de kleuren ten opzichte van de achtergrond.

Subgrid

grid-template-rows: subgrid;

Hiermee kunnen geneste elementen aansluiten op het parent grid.

View Transitions

Met View Transitions kun je een animatie toevoegen bij het navigeren naar een andere pagina. Als twee elementen dezelfde view-transition-name hebben, worden ze geanimeerd bij de overgang.
